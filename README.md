# DnD-Stories

This repo just contains material for DnD (_Dungeons and Dragons_) games, mostly for those where I am the GM (_Game Master_).
Everything I store here is homebrew (created by me).
Among this material I store:

- **Game modules**: Documents with all the data to run the games.
- **Homebrew material**: Either artifacts, characters, magic or anything that can be created.

> Note: 
> This introduction is in English, but all material is actually in Spanish.
